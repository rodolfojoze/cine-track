# 🎬 Cine-Track

API REST desenvolvida em **Kotlin + Spring Boot** para o gerenciamento de clientes, filmes e histórico de locações de uma locadora.

O projeto foi criado com foco em **boas práticas de arquitetura**, separação de responsabilidades e como material de **estudo e mentoria em Kotlin e Spring Boot**.

---

## 📌 Funcionalidades

- Cadastro e edição de clientes
- Cadastro e edição de filmes
- Controle de status de clientes e filmes
- Registro de locação de filmes
- Consulta de histórico de locações
- Tratamento global de exceções
- API estruturada em camadas (Controller, Service e Repository)

---

## 🛠 Tecnologias Utilizadas

- **Kotlin**
- **Spring Boot**
- **Spring Web**
- **Spring Data JPA**
- **Hibernate**
- **Maven**
- **Maven Wrapper**
